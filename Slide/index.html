    Exercise

    Write a shell script to set an alarm



    Instructions:

    Create a file and name it as script.sh

    $ bash script.sh 

    
#!/bin/bash
echo "Sleeping for $1 minutes"
sleep $(expr $1 \* 60)
echo -e "\a" # Alarm
echo "Time up"
